# LoVeT ClinicAI — Issue Tracker

Central issue tracker for the LoVeT veterinary clinic management system.

**[Open the Kanban Board](https://github.com/orgs/codama-dev/projects/2/views/2)**

## Creating a Ticket

Click **[New Issue](../../issues/new/choose)** and pick a template:

- **Bug Report** — something isn't working (include screenshots and console logs!)
- **Feature Request** — suggest new functionality
- **Improvement** — enhance an existing feature

## Workflow

```
Backlog → Ready for Work → In Progress → Ready for Product Test → Done
```

If a ticket fails product testing, it moves back to **Ready for Work** and the rejection count is automatically tracked.
